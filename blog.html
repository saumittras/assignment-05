A. What is the Document Object Model (DOM)?

The DOM is a JavaScript web document manipulation system. It allows access to and manipulation of web content, structure, and web pages. A DOM represents a document as a tree of nodes. The nodes represent the document's elements, attributes, and text. The key features of the DOM are its tree structure, language agnostic nature, dynamic manipulation, event handling, and content modification.

B. How do you select an element from the DOM?

There are several methods used to select an element. The selection methods are:

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

D. How do you manipulate an element's attributes and styles using the DOM?

Manupulating Attributes:

  1. Step 1:
    • Accessing an Element: getElementByClassName() or getElementById() or querySelector()

      • > example: const element = document.getElementById("ID_name");

  2. Step 2:
    • Geting an Attribute: To get an attribute, getAttribute methode is use.

      • >example: const value = element.getAttribute("attribute-name");

  3. Step 3:
    • Setting an Attribute: setAttribute methode is use for change an attribute value.

      • > example: element.setAttribute("attribute-name","NewAttributeValue");

  4. Step 4:
    • Removing an Attribute: removeAtttribute methode is use to remove an attribute.
